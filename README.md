# Robot-Software-Module
brief:
Some basic robot code written for XHU_RRC to participate in the ROBOCON competition
为西华大学机器人研究中心参加ROBOCON比赛而写的机器人基础代码
# Configuration
Step 1: Add all files to the project.
Step 2: Add include path.
Step 3: Add global macro: XHU_RRC_LIB_ENABLE.
Step 4: Configure macros in the Conf folder as needed.
# Note
Files prefixed with lib are for global configuration, while files prefixed with mod are
for the corresponding module configuration.
